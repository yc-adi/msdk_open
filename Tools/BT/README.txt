
Folder layout:
	- bin: Includes .exe files
	- daplink: Inlcudes daplink firmware (binaries) which is customized to support I2C/SPI bridge mode.
	- devices: Include device related bootloader files, test key and test images (blinkled)
	- doc: Includes bootloaders and this tool related documentations
	- src: Includes "Maxim Bootloader GUI Tool" source code, it is python based.

Please go "www.maximintegrated.com" and search your target device (Like MAX32660 or MAX32670...) 
to get all final version of documents.
